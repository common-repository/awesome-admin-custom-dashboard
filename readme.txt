== Description ==

**Awesome Admin** is an impressive WordPress custom dashboard. **Awesome Admin** provides Multiple pre-defined layout options. The best thing in this is we use Vue.js for the rendered page means **No Reload, No Refresh**.

1. Upload the plugin and activate it (alternatively, install through the WP admin console)

3. Customize the options in settings
